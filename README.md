# Bakery Website

## System requirements
  * Node.js

## Project configuration
  * Clone Back-end application: https://github.com/fcmilda/server
  * Rename .env.local-example file to .env.local
  * Define variables:
    * REACT_APP_AUTH_TOKEN_IN_LOCAL_STORAGE
    * REACT_APP_API_SERVER_ADDRESS
For EmailJS (https://dashboard.emailjs.com/sign-up)  
    * REACT_APP_EMAILJS_SERVICE_ID
    * REACT_APP_EMAILJS_TEMPLATE_ID
    * REACT_APP_EMAILJS_USER_ID

## Instalation
  * npm i

## Project scripts
  * npm start - lounch development server
  * npm build - compile app
